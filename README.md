# BookMyShow Database Design Assignment

This repository contains the database design and SQL implementation
for a BookMyShow-like movie show listing system.

The goal is to design a normalized relational database that allows
users to view all shows running in a theatre for a selected date.

---

## Features Covered
- Theatre-wise movie show listing
- Multiple screens per theatre
- Multiple shows per movie per day
- Fully normalized schema (1NF → BCNF)

---

## Folder Structure
- sql/        → Executable MySQL scripts
- diagrams/   → ER diagram 

---

## How to Run
1. Create a database in MySQL
2. Run sql/schema.sql
3. Run sql/sample-data.sql
4. Run sql/queries.sql
